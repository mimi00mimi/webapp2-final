第８〜１０回

Git、Github

怪しかったところまとめ

  • フォーク：他人のリポジトリを自分のリポジトリにコピー
  • クローン：リポジトリをローカル上にコピー
  • ローカルからGithubに変更を反映
    1. ステージング

      git add ファイル名

    2. コミット

      git commit -m "コメント"

    3. push

      git push origin main

  • 別ブランチからのマージ

    git merge 取り込むブランチ

第１１〜１５回

Docker, Node.js

怪しかったところまとめ

  • Dockerコンテナ起動：compose.yamlに必要な情報を書いた上で

    1. docker compose up -d

    2. dockercompose exec app bash

  • Dockerコンテナ一覧

    docker compose ps

  • Node.js：ブラウザ以外でのjs実行環境
  • モジュール呼び出し

    require('node:fs')

  • 比較関数 sortに渡す無名関数

    pair2[]. - pair1[].
